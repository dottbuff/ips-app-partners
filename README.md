# (db) Partners
Invision Community 4 App that adds a block with partners in a slider

## Requirements
- [IPS Community Suite: 4.7 version](https://invisioncommunity.com/)
- PHP 8.0+ - For IPS 4.7
- [GD](https://www.php.net/manual/en/book.image.php) - Image Processing and GD
